# Function: convertAmountToShare()

> **convertAmountToShare**(`aptos`, `market`, `amount`): `Promise`\<`number`\>

• **aptos**: `Aptos`

The Aptos instance from @aptos-labs/ts-sdk

• **market**: `string`

The market identifier.

• **amount**: `AnyNumber`

A promise that resolves to the converted share amount.
