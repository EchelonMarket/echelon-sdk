# Function: getAccountSupply()

> **getAccountSupply**(`aptos`, `account`, `market`): `Promise`\<`number`\>

## Parameters

• **aptos**: `Aptos`

• **account**: `string`

The address of the account.

• **market**: `string`

The market identifier.

A promise that resolves to the supplied amount.
